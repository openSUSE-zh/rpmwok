RPMWOK

Cook RPM packages of software that cannot be built easily on OBS

Add repository in openSUSE & SLE

sudo zypper ar -f https://rpmwok.guoyunhe.me/rpmwok.repo
sudo zypper ref

Add repository in Fedora & CentOS

sudo dnf config-manager --add-repo https://rpmwok.guoyunhe.me/rpmwok.repo
sudo dnf config-manager --set-enabled rpmwok
